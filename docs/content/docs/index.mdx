# Getting Started

DesKit is an Electron desktop utility for launching apps, running plugin commands,
capturing screenshots, and keeping frequently used tools close at hand.

## Floating Ball

Enable the Floating Ball in Settings to keep a small shortcut control on the
desktop. Click it to open the radial menu, then choose one of the configured
tools or plugin commands.

You can drag the Floating Ball around the screen. When you release it near the
left or right edge of the current work area, it snaps to that edge and leaves
half of the ball visible so it remains easy to reopen.
